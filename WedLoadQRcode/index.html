ดาวโหลดโปรแกรม ดาวโหลดสำรอง ! วิธีติดตั้งโปรแกรม ! วิธีใช้งานโปรแกรม

วิธีติดตั้งโปรแกรม

    1.แตกไฟล์ที่ดาวโหลด

    2.คลิ๊กเข้าไฟล์ที่แยกออกมา แล้วกดไฟล์ dist

    3.กดเปิดโปรแกรมได้เลย

วิธีใช้งานโปรแกรม

    1.วางลิ้งก์ที่ต้องการลงในช่องข้อความ

    2.กด ok แล้วจะได้ QR code ถ้ากด save จะขึ้นหน้าต่างให้ตั้งชื่อรูป ตั้งชื่อแล้วกด ok

    3.รูปจะบันทึกเข้าในแฟ้มจุดเปิดใช้งานโปรแกรม

เว็บไซด์นี้สร้างโดย พัสพล เป็นเว็บเอาไว้โหลดโปรแกรมสร้าง QR code แบบบฟรีไม่มีโฆษณาและปลอยภัย 100% วิธีที่จะเข้าเว็บนี้ได้ ต้องเข้าผ่านลิ้งก์เท่านั้นหรือไปที่หน้าโปรไฟล์เฟซบุ๊ค พัสพล พลังชีวิต แล้วกดผ่านลิ้งก์ ถึงจะสามารถเข้าใช้งานเว็บไซด์นี้ได้
